# Library Management System

## Introduction
The Library Management System (LMS) is a web-based application designed to streamline the management of library resources. It offers functionalities such as borrowing and returning books, cataloging new arrivals, managing member registrations, and tracking late returns.

## Features
- **User Management**: Manage user accounts for library members.
- **Book Catalog**: Search and browse the library's collection of books.
- **Borrowing System**: Members can borrow books and keep track of due dates.
- **Return Management**: Easily manage book returns and calculate fines for late returns.
- **Admin Dashboard**: Administrative tools for managing books, members, and transactions.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/DarshitThakur/Library-Management-System.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Library-Management-System
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Usage
Once the application is running, you can access it through your web browser at `http://localhost:3000`. Create an account or log in to start managing your library.

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express
- **Database**: MongoDB
